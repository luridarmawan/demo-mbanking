## Cara instalasi 

Cara instalasi atau build dari repo ini cukup mudah:.

1. Unduk repo ini, atau bisa dengan git pull
2. jalankan `flutter run`
    misal:
    ```bash
    flutter -d [iddevice] run
    ```
